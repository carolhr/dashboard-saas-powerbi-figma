# 🚀 Dashboard Avançado SaaS: Power BI + Figma (UI/UX)

# 📌 Sobre o Projeto
Este repositório contém um dashboard interativo focado na Saúde Financeira de um modelo de negócio baseado em Assinaturas (SaaS/Recorrência). O projeto foi desenvolvido com o objetivo de unir conceitos avançados de Engenharia de Dados (DAX) com práticas modernas de Design de Interface (UI/UX).

# 🛠️ Tecnologias e Ferramentas Utilizadas
Power BI Desktop: Modelagem de dados, criação de métricas e construção dos gráficos.
Figma: Desenho da arquitetura de blocos, sombras projetadas, cantos arredondados e conceito de Dark Mode de alta fidelidade.
Microsoft Excel: Estruturação da base de dados fictícia utilizada no projeto.

# 🧮 Fórmulas e Métricas Desenvolvidas (DAX)
O projeto conta com inteligência de negócio aplicada através de medidas calculadas do zero:
MRR (Monthly Recurring Revenue): Soma da receita recorrente mensal considerando exclusivamente a volumetria de usuários qualificados com o status ativo.
Clientes Ativos: Contagem dinâmica e contextualizada da base de clientes recorrentes.
Crescimento Mensal % (MoM): Cálculo fracionado utilizando funções de inteligência de tempo para comparar a evolução de receita entre meses consecutivos de forma automatizada.

# 🎨 Diferenciais de UI/UX (Experiência do Usuário)
Dark Mode Premium: Escolha de paleta de cores balanceada (grafite e verde neon) para evitar a fadiga visual e direcionar a atenção aos KPIs críticos.
Layout Fluído: Fundo totalmente estruturado no Figma e importado como background para otimização de memória e alinhamento geométrico perfeito dos visuais.
